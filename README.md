# 🔍 Palindrome Checker App

## 📌 Project Overview

The **Palindrome Checker App** is a console-based Java application designed to check whether a given string is a palindrome using multiple algorithms and data structures.

This project demonstrates core Java programming concepts, clean Git workflow practices, and efficient use of fundamental data structures.

---

## 🚀 Features (Use Cases)

| Use Case | Description | Data Structure Used |
|---------|-------------|---------------------|
| UC1 | Application startup and welcome message | None |
| UC2 | Hardcoded palindrome check | String |
| UC3 | Reverse string method | String |
| UC4 | Character array comparison | char[] |
| UC5 | Stack-based palindrome check | Stack |
| UC6 | Queue and Stack comparison | Queue, Stack |
| UC7 | Optimized palindrome check | Deque |

---
